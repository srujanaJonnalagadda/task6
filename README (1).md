 Project Overview
This project demonstrates how to work with an online sales dataset using MySQL.
It covers:

Table creation & data insertion

Extracting month/year from dates

Grouping & aggregating sales data

Calculating revenue & order volumes

Sorting & filtering results for analysis

The dataset contains 104 orders from Jan 2024 to Apr 2024, with the following columns:

order_date – Date of the order

amount – Order amount in USD

product_id – Unique product identifier
